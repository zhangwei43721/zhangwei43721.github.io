ж•°жҚ®еә“ж ҮзәўеҶ…е®№ж‘ҳеҪ•

з®Җиҝ°ж•°жҚ®еә“зҡ„дёүзә§жЁЎејҸз»“жһ„пјҲдё»иҰҒиғҢжЁЎејҸзҡ„жҰӮеҝөпјүгҖӮ

еҜ№з”ЁжҲ·иҖҢиЁҖпјҢеӨ–жЁЎејҸгҖҒжЁЎејҸе’ҢеҶ…жЁЎејҸеҲҶеҲ«еҜ№еә”дёҖиҲ¬з”ЁжҲ·жЁЎејҸгҖҒжҰӮеҝөжЁЎејҸе’Ңзү©зҗҶжЁЎејҸпјҢе®ғд»¬еҲҶеҲ«еҸҚжҳ дәҶзңӢеҫ…ж•°жҚ®еә“зҡ„дёүдёӘи§’еәҰгҖӮ

жЁЎејҸд№ҹз§°дёәжҰӮеҝөжЁЎејҸпјҢжҳҜж•°жҚ®еә“дёӯе…ЁдҪ“ж•°жҚ®зҡ„йҖ»иҫ‘з»“жһ„е’Ңзү№еҫҒзҡ„жҸҸиҝ°пјҢеӨ„дәҺдёүзә§жЁЎејҸз»“жһ„зҡ„дёӯй—ҙеұӮпјҢдёҚж¶үеҸҠж•°жҚ®зҡ„зү©зҗҶеӯҳеӮЁз»ҶиҠӮе’ҢзЎ¬д»¶зҺҜеўғпјҢдёҺе…·дҪ“зҡ„еә”з”ЁзЁӢеәҸгҖҒжүҖдҪҝз”Ёзҡ„еә”з”ЁејҖеҸ‘е·Ҙе…·еҸҠй«ҳзә§зЁӢеәҸи®ҫи®ЎиҜӯиЁҖж— е…ігҖӮдёҖдёӘж•°жҚ®еә“еҸӘжңүдёҖдёӘжЁЎејҸгҖӮ

пјҲе°ұжҳҜжҸҸиҝ°еӨ–жЁЎејҸе’ҢеҶ…жЁЎејҸзҡ„жҳ е°„е…ізі»зҡ„йғЁеҲҶпјҢеҰӮдҪ е®ҡд№үдёҖеј еӯҳдәҶеӯҰз”ҹйҖүиҜҫзҡ„дҝЎжҒҜиЎЁпјү

еӨ–жЁЎејҸеҸҲз§°дёәеӯҗжЁЎејҸжҲ–з”ЁжҲ·жЁЎејҸпјҢжҳҜдёүзә§жЁЎејҸз»“жһ„зҡ„жңҖеӨ–еұӮпјҢжҳҜдёҺжҹҗдёҖеә”з”Ёжңүе…ізҡ„ж•°жҚ®зҡ„йҖ»иҫ‘з»“жһ„пјҢеҚіз”ЁжҲ·и§ҶеӣҫгҖӮеӨ–жЁЎејҸдёҖиҲ¬жҳҜжЁЎејҸзҡ„еӯҗйӣҶпјҢдёҖдёӘж•°жҚ®еә“еҸҜд»ҘжңүеӨҡдёӘеӨ–жЁЎејҸгҖӮ

пјҲе°ұжҳҜж•°жҚ®еә“з”ЁжҲ·пјҲеҢ…жӢ¬еә”з”ЁзЁӢеәҸе‘ҳе’ҢжңҖз»Ҳз”ЁжҲ·пјүжңҖз»ҲеҸҜд»ҘзңӢеҲ°зҡ„еҶ…е®№пјҢеҰӮдҪ з»ҷиҖҒеёҲе’ҢеӯҰз”ҹеҲҶеҲ«е®ҡд№үдәҶд»–д»¬иғҪзңӢйҖүиҜҫиЎЁзҡ„е“ӘдәӣеҶ…е®№пјү

еҶ…жЁЎејҸеҸҲз§°еӯҳеӮЁжЁЎејҸжҲ–зү©зҗҶжЁЎејҸпјҢжҳҜдёүзә§жЁЎејҸз»“жһ„дёӯзҡ„жңҖеҶ…еұӮпјҢд№ҹжҳҜйқ иҝ‘зү©зҗҶеӯҳеӮЁзҡ„дёҖеұӮпјҢеҚідёҺе®һйҷ…еӯҳеӮЁж•°жҚ®ж–№ејҸжңүе…ізҡ„дёҖеұӮгҖӮе®ғжҳҜеҜ№ж•°жҚ®еә“еӯҳеӮЁз»“жһ„зҡ„жҸҸиҝ°пјҢжҳҜж•°жҚ®еңЁж•°жҚ®еә“еҶ…йғЁзҡ„иЎЁзӨәж–№ејҸгҖӮдёҖдёӘж•°жҚ®еә“еҸӘжңүдёҖдёӘеҶ…жЁЎејҸгҖӮ

пјҲж•°жҚ®еңЁж•°жҚ®еә“дёҠеӯҳеӮЁзҡ„еҶ…е®№пјҢжҳҜж•°жҚ®еңЁж•°жҚ®еә“еҶ…йғЁзҡ„иЎЁзӨәж–№ејҸпјҢеҰӮдҪ е®ҡд№үеӯҰз”ҹйҖүиҜҫдҝЎжҒҜиЎЁзҡ„ж•°жҚ®жҢүз…§еӯҰз”ҹidзҡ„B+ж ‘зҙўеј•еӯҳеӮЁпјүпјҲдҫӢеҰӮпјҢи®°еҪ•зҡ„еӯҳеӮЁж–№ејҸжҳҜйЎәеәҸеӯҳеӮЁгҖҒжҢүз…§Bж ‘з»“жһ„еӯҳеӮЁиҝҳжҳҜжҢүhashж–№жі•еӯҳеӮЁпјӣзҙўеј•жҢүз…§д»Җд№Ҳж–№ејҸз»„з»Үпјӣж•°жҚ®жҳҜеҗҰеҺӢзј©еӯҳеӮЁпјҢжҳҜеҗҰеҠ еҜҶпјӣж•°жҚ®зҡ„еӯҳеӮЁи®°еҪ•з»“жһ„жңүдҪ•и§„е®ҡпјүгҖӮ

дёүдёӘдё–з•ҢжҢҮд»Җд№Ҳпјҹе®ғд»¬д№Ӣй—ҙжңүд»Җд№ҲиҒ”зі»пјҹ

дёүдёӘдё–з•ҢжҳҜжҢҮзҺ°е®һдё–з•ҢгҖҒдҝЎжҒҜдё–з•Ңе’Ңи®Ўз®—жңәдё–з•ҢгҖӮ

  • зҺ°е®һдё–з•ҢпјҢеҚіе®ўи§ӮеӯҳеңЁзҡ„дё–з•ҢгҖӮе…¶дёӯеӯҳеңЁзқҖеҗ„з§ҚдәӢзү©еҸҠе®ғд»¬д№Ӣй—ҙзҡ„иҒ”зі»пјҢжҜҸдёӘдәӢзү©йғҪжңүиҮӘе·ұзҡ„зү№еҫҒжҲ–жҖ§иҙЁгҖӮ
  • дҝЎжҒҜдё–з•ҢжҳҜзҺ°е®һдё–з•ҢеңЁдәәд»¬еӨҙи„‘дёӯзҡ„еҸҚжҳ пјҢз»ҸиҝҮдәәи„‘зҡ„еҲҶжһҗгҖҒеҪ’зәіе’ҢжҠҪиұЎпјҢеҪўжҲҗдҝЎжҒҜпјҢдәәд»¬жҠҠиҝҷдәӣдҝЎжҒҜиҝӣиЎҢи®°еҪ•гҖҒж•ҙзҗҶгҖҒеҪ’зұ»е’Ңж јејҸеҢ–еҗҺпјҢе°ұжһ„жҲҗдәҶдҝЎжҒҜдё–з•ҢгҖӮ
  • и®Ўз®—жңәдё–з•ҢжҳҜдҝЎжҒҜдё–з•ҢдёӯдҝЎжҒҜзҡ„ж•°жҚ®еҢ–пјҢе°ұжҳҜе°ҶдҝЎжҒҜз”Ёеӯ—з¬Ұе’Ңж•°еҖјзӯүж•°жҚ®иЎЁзӨәпјҢдҫҝдәҺеӯҳеӮЁеңЁи®Ўз®—жңәдёӯе№¶з”ұи®Ўз®—жңәиҝӣиЎҢиҜҶеҲ«е’ҢеӨ„зҗҶгҖӮ

еҜ№дёҠйқўеәҹиҜқзҡ„жҖ»з»“пјҡзҺ°е®һдё–з•Ңзҡ„е®ўи§ӮдәӢзү©вҶ’дҝЎжҒҜдё–з•ҢпјҲеӨ§и„‘пјүзҡ„жҠҪиұЎе®һдҪ“еұһжҖ§вҶ’жңәеҷЁдё–з•Ңзҡ„ж•°жҚ®и®°еҪ•

дёҫдҫӢиҜҙжҳҺд»Җд№ҲжҳҜе®һдҪ“е®Ңж•ҙжҖ§е’ҢеҸӮз…§е®Ңж•ҙжҖ§гҖӮ

е®һдҪ“е®Ңж•ҙжҖ§жҳҜжҢҮдё»з Ғзҡ„еҖјдёҚиғҪдёәз©әжҲ–йғЁеҲҶдёәз©әгҖӮ

дҫӢеҰӮпјҢж•ҷжқҗдёӯиЎЁ1-3зҡ„иҜҫзЁӢе…ізі»дёӯзҡ„дё»з ҒвҖңиҜҫзЁӢеҸ·вҖқпјҲcnoпјүеҸҜд»Ҙе”ҜдёҖж ҮиҜҶдёҖй—ЁиҜҫзЁӢе®һдҪ“гҖӮеҰӮжһңдё»з Ғдёӯзҡ„еҖјдёәз©әжҲ–йғЁеҲҶдёәз©әпјҢеҚідё»еұһжҖ§дёәз©әпјҢеҲҷдёҚз¬ҰеҗҲе…ізі»й”®зҡ„е®ҡд№үжқЎд»¶пјҢдёҚиғҪе”ҜдёҖж ҮиҜҶе…ғз»„еҸҠдёҺе…¶зӣёеҜ№еә”зҡ„е®һдҪ“гҖӮдҫӢеҰӮпјҢиҜҫзЁӢе…ізі»дёӯзҡ„дё»з ҒвҖңиҜҫзЁӢеҸ·вҖқдёҚиғҪдёәз©әпјҢжҺҲиҜҫе…ізі»дёӯзҡ„дё»з ҒвҖңж•ҷеёҲеҸ·+иҜҫзЁӢеҸ·вҖқдёҚиғҪйғЁеҲҶдёәз©әпјҢеҚівҖңж•ҷеёҲеҸ·вҖқе’ҢвҖңиҜҫзЁӢеҸ·вҖқдёӨдёӘеӯ—ж®өзҡ„еҸ–еҖјйғҪдёҚиғҪдёәз©әгҖӮ

  • пјҲж–ҮжЎЈйҮҢзҡ„еҺҹиҜқпјҢзңӢдёҚжҮӮжҖқеҜҶиҫҫ~пјүеҸӮз…§е®Ңж•ҙжҖ§жҳҜжҢҮеҰӮжһңе…ізі»R2зҡ„еӨ–з ҒXдёҺе…ізі»R1зҡ„дё»з Ғзӣёз¬ҰпјҢеҲҷXзҡ„жҜҸдёӘеҖјжҲ–иҖ…зӯүдәҺR1дёӯдё»з Ғзҡ„жҹҗдёҖдёӘеҖјжҲ–иҖ…еҸ–з©әеҖјгҖӮ
    • и§ЈйҮҠиҝҷеҸҘиҜқпјҡ
      • е…ізі»R1пјҡиҝҷжҳҜдёҖдёӘе…·жңүдё»з ҒпјҲPrimary Keyпјүзҡ„е…ізі»пјҲиЎЁпјүпјҲдё»иЎЁпјүгҖӮдё»з ҒпјҲдё»й”®пјүжҳҜе…ізі»дёӯе”ҜдёҖж ҮиҜҶжҜҸдёҖиЎҢзҡ„еұһжҖ§жҲ–еұһжҖ§з»„еҗҲгҖӮдҫӢеҰӮпјҢеӯҰз”ҹиЎЁдёӯзҡ„еӯҰеҸ·еҸҜд»ҘдҪңдёәдё»з ҒгҖӮ
      • е…ізі»R2пјҡиҝҷжҳҜеҸҰдёҖдёӘе…ізі»пјҲиЎЁпјүпјҢе…¶дёӯеҢ…еҗ«дёҖдёӘеӨ–з ҒпјҲForeign KeyпјүпјҲд»ҺиЎЁпјүгҖӮеӨ–з ҒпјҲеӨ–й”®пјүжҳҜе…ізі»дёӯзҡ„дёҖдёӘеұһжҖ§пјҢе®ғдёҺе…¶д»–е…ізі»зҡ„дё»з Ғзӣёе…іиҒ”гҖӮдҫӢеҰӮпјҢйҖүдҝ®иҜҫиЎЁдёӯзҡ„иҜҫзЁӢзј–еҸ·еҸҜд»ҘдҪңдёәеӨ–з ҒпјҢдёҺеӯҰз”ҹиЎЁдёӯзҡ„еӯҰеҸ·е…іиҒ”гҖӮ
      • еҰӮжһңе…ізі»R2зҡ„еӨ–з ҒXдёҺе…ізі»R1зҡ„дё»з Ғзӣёз¬ҰпјҢж„Ҹе‘ізқҖXзҡ„жҜҸдёӘеҖјйғҪеҝ…йЎ»ж»Ўи¶ід»ҘдёӢжқЎд»¶пјҡ
        1. иҰҒд№ҲзӯүдәҺR1дёӯжҹҗдёҖиЎҢзҡ„дё»з ҒеҖјгҖӮ
        2. иҰҒд№Ҳдёәз©әеҖјпјҲеҚіжІЎжңүеҜ№еә”зҡ„дё»з ҒеҖјпјүгҖӮ
  • з”ЁдәәиҜқи§ЈйҮҠдёҠйқўзҡ„еҶ…е®№пјҡеҰӮжһңеңЁеӯҰз”ҹиЎЁе’ҢйҖүдҝ®иҜҫд№Ӣй—ҙз”ЁеӯҰеҸ·е»әз«Ӣе…іиҒ”пјҢеӯҰз”ҹиЎЁжҳҜдё»иЎЁпјҢйҖүдҝ®иҜҫжҳҜд»ҺиЎЁпјҢйӮЈд№ҲпјҢеңЁеҗ‘йҖүдҝ®иҜҫиЎЁпјҲд»ҺиЎЁпјүдёӯиҫ“е…ҘдёҖжқЎж–°и®°еҪ•ж—¶пјҢзі»з»ҹиҰҒжЈҖжҹҘж–°и®°еҪ•зҡ„еӯҰеҸ·жҳҜеҗҰеңЁеӯҰз”ҹиЎЁпјҲдё»иЎЁпјүдёӯе·ІеӯҳеңЁпјҢеҰӮжһңеӯҳеңЁпјҢеҲҷе…Ғи®ёжү§иЎҢиҫ“е…Ҙж“ҚдҪңпјҢеҗҰеҲҷжӢ’з»қиҫ“е…ҘпјҢиҝҷе°ұжҳҜеҸӮз…§е®Ңж•ҙжҖ§гҖӮ

з®Җиҝ°зӣёе…іеӯҗжҹҘиҜўзҡ„иҝҮзЁӢгҖӮ

зӣёе…іеӯҗжҹҘиҜўзҡ„жү§иЎҢйЎәеәҸжҳҜпјҡ

йҰ–е…ҲпјҢйҖүеҸ–зҲ¶жҹҘиҜўиЎЁдёӯзҡ„з¬¬дёҖиЎҢи®°еҪ•пјҢеӯҗжҹҘиҜўеҲ©з”ЁжӯӨиЎҢдёӯзӣёе…ізҡ„еұһжҖ§еҖјеңЁеӯҗжҹҘиҜўи®ҫи®Ўзҡ„еҹәжң¬иЎЁдёӯиҝӣиЎҢжҹҘиҜўпјҢз„¶еҗҺпјҢзҲ¶жҹҘиҜўж №жҚ®еӯҗжҹҘиҜўиҝ”еӣһзҡ„з»“жһңеҲӨж–ӯзҲ¶жҹҘиҜўиЎЁдёӯзҡ„жӯӨиЎҢжҳҜеҗҰж»Ўи¶іжҹҘиҜўжқЎд»¶гҖӮеҰӮжһңж»Ўи¶іжқЎд»¶пјҢеҲҷжҠҠиҜҘиЎҢж”ҫе…ҘзҲ¶жҹҘиҜўзҡ„жҹҘиҜўз»“жһңйӣҶеҗҲдёӯгҖӮйҮҚеӨҚжү§иЎҢиҝҷдёҖиҝҮзЁӢпјҢзӣҙеҲ°еӨ„зҗҶе®ҢзҲ¶жҹҘиҜўиЎЁдёӯзҡ„жҜҸдёҖиЎҢж•°жҚ®гҖӮпјҲиҝҷеҸҘеәҹиҜқеӨӘй•ҝпјҢзӣҙжҺҘзңӢдёӢйқўзҡ„дҫӢеӯҗзҗҶи§Јпјү

дёҫдёӘдҫӢеӯҗпјҢ еҒҮи®ҫжҲ‘д»¬жңүдёҖдёӘеҗҚдёә employees зҡ„иЎЁпјҢе…¶дёӯеҢ…еҗ«е‘ҳе·Ҙзҡ„дҝЎжҒҜпјҲдҫӢеҰӮе‘ҳе·Ҙ IDгҖҒе§“еҗҚгҖҒи–Әж°ҙзӯүпјүгҖӮжҲ‘д»¬жғіиҰҒжүҫеҮәи–Әж°ҙй«ҳдәҺжүҖжңүе‘ҳе·Ҙе№іеқҮи–Әж°ҙзҡ„е‘ҳе·ҘгҖӮд»ҘдёӢжҳҜжҹҘиҜўиҜӯеҸҘпјҡ

иҝҷе°Ҷиҝ”еӣһи–Әж°ҙй«ҳдәҺе№іеқҮи–Әж°ҙзҡ„е‘ҳе·ҘеҲ—иЎЁгҖӮ

и§Ҷеӣҫзҡ„дҪңз”Ё

жҸҗеҚҮж•°жҚ®ж“ҚдҪңзҡ„дҫҝжҚ·жҖ§гҖҒжҸҗеҚҮж•°жҚ®зҡ„йҖ»иҫ‘зӢ¬з«ӢжҖ§гҖҒжҸҗеҚҮж•°жҚ®зҡ„е®үе…ЁжҖ§гҖҒж•°жҚ®йӣҶжҲҗгҖӮ

д»ҘдёӢжҳҜе°ҶеӣҫзүҮдёӯзҡ„еҶ…е®№зӣҙжҺҘиҪ¬жҚўдёә Markdown ж јејҸзҡ„дёӯж–ҮзүҲжң¬пјҢдҝқз•ҷдәҶеҺҹж–Үзҡ„ж•°еӯҰз¬ҰеҸ·е’Ңж јејҸпјҡ

еҮҪж•°дҫқиө–

жҳҜе…ізі»жЁЎејҸдёӯеұһжҖ§д№Ӣй—ҙзҡ„дёҖз§ҚйҖ»иҫ‘дҫқиө–е…ізі»

и®ҫжңүе…ізі»жЁЎејҸ ( R(U) )пјҢ( U ) жҳҜеұһжҖ§е…ЁйӣҶпјҢ( X ) е’Ң ( Y ) жҳҜ ( U ) зҡ„еӯҗйӣҶпјҢеҰӮжһң ( X \rightarrow Y )пјҢе№¶дё”еҜ№дәҺ ( X ) зҡ„д»»дҪ•дёҖдёӘзңҹеӯҗйӣҶ ( XвҖҷ )пјҢйғҪжңү ( XвҖҷ \not\rightarrow Y )пјҢеҲҷз§° ( Y ) еҜ№ ( X ) е®Ңе…ЁеҮҪж•°дҫқиө–пјҲFull-Functional-DependencyпјүпјҢи®°дҪңпјҡ

[
X \xrightarrow{f} Y
]

еҰӮжһңеӯҳеңЁ ( X ) зҡ„жҹҗдёҖдёӘзңҹеӯҗйӣҶ ( XвҖҷ )пјҢжңү ( XвҖҷ \rightarrow Y )пјҢеҲҷз§° ( Y ) еҜ№ ( X ) йғЁеҲҶеҮҪж•°дҫқиө–пјҲPartial-Functional-DependencyпјүпјҢи®°дҪңпјҡ

[
X \xrightarrow{p} Y
]

и®ҫжңүе…ізі»жЁЎејҸ ( R(U) )пјҢ( U ) жҳҜеұһжҖ§е…ЁйӣҶпјҢ( X, Y, Z ) жҳҜ ( U ) зҡ„еӯҗйӣҶпјҢиӢҘ ( X \rightarrow Y )пјҢдҪҶжҳҜ ( Y \not\rightarrow X )пјҢиҖҢдё” ( Y \rightarrow Z )пјҲ( Y \in X, Z \in Y )пјүпјҢеҲҷз§° ( Z ) еҜ№ ( X ) дј йҖ’еҮҪж•°дҫқиө–пјҲTransitive-Functional-DependencyпјүпјҢи®°дҪңпјҡ

[
X \xrightarrow{t} Z
]

иҜ•иҝ°E-RеӣҫиҪ¬жҚўжҲҗе…ізі»жЁЎеһӢзҡ„иҪ¬жҚўи§„еҲҷгҖӮ

зӯ”пјҡE-Rеӣҫеҗ‘е…ізі»жЁЎеһӢзҡ„иҪ¬жҚўдёҖиҲ¬еә”йҒөеҫӘеҰӮдёӢеҺҹеҲҷпјҡ

пјҲ1пјүе®һдҪ“зҡ„иҪ¬жҚўгҖӮдёҖдёӘе®һдҪ“еһӢиҪ¬жҚўдёәдёҖдёӘе…ізі»жЁЎејҸгҖӮе®һдҪ“зҡ„еұһжҖ§е°ұжҳҜе…ізі»зҡ„еұһжҖ§пјҢе®һдҪ“зҡ„з Ғе°ұжҳҜе…ізі»зҡ„з ҒгҖӮ

пјҲ2пјүиҒ”зі»зҡ„иҪ¬жҚўгҖӮдёҖдёӘиҒ”зі»иҪ¬еҢ–дёәдёҖдёӘе…ізі»жЁЎејҸпјҢдёҺиҜҘиҒ”зі»зӣёиҝһзҡ„еҗ„е®һдҪ“зҡ„з Ғд»ҘеҸҠиҒ”зі»зҡ„еұһжҖ§иҪ¬еҢ–дёәе…ізі»зҡ„еұһжҖ§пјҢиҜҘе…ізі»зҡ„з ҒеҲҷжңүеҮ з§Қжғ…еҶөпјҡ

  • иӢҘиҒ”зі»дёә1:1пјҢеҲҷжҜҸдёӘе®һдҪ“зҡ„з ҒеқҮжҳҜиҜҘе…ізі»зҡ„еҖҷйҖүз ҒгҖӮ
  • иӢҘиҒ”зі»дёә1:nпјҢеҲҷе…ізі»зҡ„з Ғдёәnз«Ҝе®һдҪ“зҡ„з ҒгҖӮ
  • иӢҘиҒ”зі»дёәm:nпјҢеҲҷе…ізі»зҡ„з ҒдёәиҜёе®һдҪ“з Ғзҡ„з»„еҗҲгҖӮ

вҖӢ дёүдёӘжҲ–дёүдёӘд»ҘдёҠе®һдҪ“й—ҙзҡ„еӨҡе…ғиҒ”зі»гҖҒеҗҢдёҖе®һдҪ“йӣҶеҶ…зҡ„иҮӘеҸҚиҒ”зі»зҡ„иҪ¬жҚўи§„еҲҷдёҺдәҢе…ғиҒ”зі»зӣёеҗҢгҖӮ

пјҲ3пјүе…·жңүзӣёеҗҢз Ғзҡ„е…ізі»жЁЎејҸеҸҜеҗҲе№¶гҖӮ

дёӢйқўдёәдәҶж–№дҫҝзҗҶи§ЈпјҢдёҫдёҖдәӣдҫӢеӯҗпјҡ

дёҖгҖҒйҰ–е…ҲдәҶи§ЈE-Rеӣҫдёӯеёёз”ЁеӣҫеҪўеҸҠд»ЈиЎЁзҡ„ж„ҸжҖқ

вҖӢ 1.1 й•ҝж–№еҪўпјҡиЎЁзӨәе®һдҪ“гҖӮ

вҖӢ е®һдҪ“жҳҜе®ўи§ӮеӯҳеңЁзҡ„дәӢзү©гҖӮ

вҖӢ дҫӢеҰӮз”ЁжҲ·гҖҒе•Ҷе“ҒгҖҒи®ўеҚ•гҖҒдҫӣеә”е•ҶзӯүгҖӮиҜҙзӣҙзҷҪдәҶпјҢе°ұжҳҜж•°жҚ®еә“зҡ„иЎЁеҗҚе°ұеҸҜд»ҘеҒҡдёҖдёӘе®һдҪ“еҜ№иұЎгҖӮдёҖдёӘзі»з»ҹжҳҜз”ұеҫҲеӨҡдёӘе®һдҪ“еҜ№иұЎжһ„жҲҗзҡ„пјҢз„¶еҗҺе®ғд»¬д№Ӣй—ҙеӯҳеңЁдёҖе®ҡзҡ„е…ізі»е’ҢеұһжҖ§гҖӮ

вҖӢ 1.2 жӨӯеңҶеҪўпјҡиЎЁзӨәеұһжҖ§гҖӮ

вҖӢ еұһжҖ§еҲҶдёәе”ҜдёҖеұһжҖ§пјҲ unique attributeпјүе’Ңйқһе”ҜдёҖеұһжҖ§пјҢе”ҜдёҖеұһжҖ§жҢҮзҡ„жҳҜе”ҜдёҖеҸҜз”ЁжқҘж ҮиҜҶиҜҘе®һдҪ“е®һдҫӢжҲ–иҖ…жҲҗе‘ҳзҡ„еұһжҖ§пјҢз”ЁдёӢеҲ’зәҝиЎЁзӨәпјҢдёҖиҲ¬жқҘи®Іе®һдҪ“йғҪиҮіе°‘жңүдёҖдёӘе”ҜдёҖеұһжҖ§гҖӮ

вҖӢ дёҫдёӘж —еӯҗпјҡз”ЁжҲ·е®һдҪ“жӢҘжңүеұһжҖ§гҖҗidпјҲе”ҜдёҖеұһжҖ§пјүгҖҒе§“еҗҚгҖҒе№ҙйҫ„гҖҒз”өиҜқгҖҒиә«д»ҪиҜҒеҸ·пјҲе”ҜдёҖеұһжҖ§пјүгҖ‘гҖҒе•Ҷе“Ғе®һдҪ“жӢҘжңүеұһжҖ§гҖҗidгҖҒе•Ҷе“ҒеҗҚз§°гҖҒе•Ҷе“Ғзұ»еһӢгҖҒе•Ҷе“Ғд»·ж јгҖҒе•Ҷе“ҒеӣҫзүҮгҖҒе•Ҷе“ҒжҸҸиҝ°гҖҒдҫӣеә”е•ҶеҗҚз§°гҖ‘гҖҒи®ўеҚ•е®һдҪ“жӢҘжңүеұһжҖ§гҖҗidгҖҒи®ўеҚ•зј–еҸ·гҖҒз”ЁжҲ·еҗҚз§°гҖҒе•Ҷе“ҒеҗҚз§°гҖҒдёӢеҚ•ж—ҘжңҹгҖ‘гҖҒдҫӣеә”е•Ҷе®һдҪ“жӢҘжңүеұһжҖ§гҖҗidгҖҒеҗҚз§°гҖҒең°еқҖгҖҒиҒ”зі»з”өиҜқгҖ‘гҖӮ

вҖӢ 1.3 иҸұеҪўпјҡиЎЁзӨәе…ізі»гҖӮ

вҖӢ дҫӢеҰӮз”ЁжҲ·е’Ңе•Ҷе“Ғеә”иҜҘиҙӯд№°е…ізі»пјҲдёҖдёӘз”ЁжҲ·иҙӯд№°еӨҡд»¶е•Ҷе“ҒпјүгҖҒи®ўеҚ•дёҺе•Ҷе“Ғеә”иҜҘжҳҜеҢ…еҗ«е…ізі»пјҲдёҖдёӘи®ўеҚ•дёӯеҢ…еҗ«еӨҡд»¶е•Ҷе“ҒпјүгҖӮ

дәҢгҖҒERеӣҫдёӯе…іиҒ”е…ізі»жңүдёүз§Қ

вҖӢ **дёҖеҜ№дёҖпјҲ1:1пјү пјҡ**1еҜ№1е…ізі»жҳҜжҢҮеҜ№дәҺе®һдҪ“йӣҶAдёҺе®һдҪ“йӣҶBпјҢAдёӯзҡ„жҜҸдёҖдёӘе®һдҪ“иҮіеӨҡдёҺBдёӯдёҖдёӘе®һдҪ“жңүе…ізі»пјӣеҸҚд№ӢпјҢеңЁе®һдҪ“йӣҶBдёӯзҡ„жҜҸдёӘе®һдҪ“иҮіеӨҡдёҺе®һдҪ“йӣҶAдёӯдёҖдёӘе®һдҪ“жңүе…ізі»гҖӮ

вҖӢ дҫӢеҰӮпјҡдёҖдёӘз”ЁжҲ·еҸӘиғҪжӢҘжңүдёҖеј иә«д»ҪиҜҒпјҢиҖҢдёҖеј иә«д»ҪиҜҒеҸӘеұһдәҺдёҖдёӘз”ЁжҲ·гҖӮжүҖд»Ҙиҝҷе°ұжҳҜдёҖеҜ№дёҖзҡ„е…ізі»гҖӮ

вҖӢ **дёҖеҜ№еӨҡпјҲ1:nпјү пјҡ**1еҜ№еӨҡе…ізі»жҳҜжҢҮе®һдҪ“йӣҶAдёҺе®һдҪ“йӣҶBдёӯиҮіе°‘жңүn(n>0)дёӘе®һдҪ“жңүе…ізі»пјӣе№¶дё”е®һдҪ“йӣҶBдёӯжҜҸдёҖдёӘе®һдҪ“иҮіеӨҡдёҺе®һдҪ“йӣҶAдёӯдёҖдёӘе®һдҪ“жңүе…ізі»гҖӮ

вҖӢ дҫӢеҰӮпјҡдёҖеҜ№еӨҡе’ҢеӨҡеҜ№дёҖжҳҜдёҖж ·зҡ„гҖӮдёҖдёӘз”ЁжҲ·жӢҘжңүеӨҡеј й“¶иЎҢеҚЎпјҢдҪҶжҳҜдёҖеј й“¶иЎҢеҚЎеҸӘеұһдәҺдёҖдёӘз”ЁжҲ·гҖӮжүҖд»Ҙиҝҷе°ұжҳҜдёҖеҜ№еӨҡзҡ„е…ізі»гҖӮеҸҚиҝҮжқҘиҜҙжі•е°ұжҳҜеӨҡеҜ№дёҖгҖӮ

вҖӢ **еӨҡеҜ№еӨҡпјҲm:nпјү пјҡ**еӨҡеҜ№еӨҡе…ізі»жҳҜжҢҮе®һдҪ“йӣҶAдёӯзҡ„жҜҸдёҖдёӘе®һдҪ“дёҺе®һдҪ“йӣҶBдёӯиҮіе°‘жңүm(m>0)дёӘе®һдҪ“жңүе…ізі»пјҢе№¶дё”е®һдҪ“йӣҶBдёӯзҡ„жҜҸдёҖдёӘе®һдҪ“дёҺе®һдҪ“йӣҶAдёӯзҡ„иҮіе°‘nпјҲn>0пјүдёӘе®һдҪ“жңүе…ізі»гҖӮ

вҖӢ дҫӢеҰӮпјҡз”ЁжҲ·дёҺе•Ҷе“Ғзҡ„е…ізі»пјҢдёҖдёӘз”ЁжҲ·еҸҜжӢҘжңүеӨҡд»¶е•Ҷе“ҒгҖӮеҗҢж ·дёҖд»¶е•Ҷе“ҒеҸҜиў«еӨҡдёӘз”ЁжҲ·жүҖжӢҘжңүгҖӮжүҖд»Ҙиҝҷе°ұжҳҜеӨҡеҜ№еӨҡзҡ„е…ізі»гҖӮ

е®һдҪ“зҡ„иҪ¬жҚў

жҜҸдёӘе®һдҪ“еһӢиҪ¬жҚўдёәдёҖдёӘе…ізі»жЁЎејҸгҖӮдҫӢеҰӮпјҢжҲ‘д»¬жңүдёҖдёӘеӯҰз”ҹе®һдҪ“пјҲStudentпјүе’ҢдёҖдёӘиҜҫзЁӢе®һдҪ“пјҲCourseпјүгҖӮ

иҪ¬жҚўеҗҺзҡ„е…ізі»жЁЎејҸпјҡ

  • еӯҰз”ҹ(еӯҰеҸ·, е§“еҗҚ, е№ҙйҫ„)
  • иҜҫзЁӢ(иҜҫзЁӢеҸ·, иҜҫзЁӢеҗҚз§°, еӯҰеҲҶ)

е…·жңүзӣёеҗҢз Ғзҡ„е…ізі»жЁЎејҸеҸҜеҗҲе№¶

еҰӮжһңдёӨдёӘе…ізі»жЁЎејҸе…·жңүзӣёеҗҢзҡ„з ҒпјҢе®ғд»¬еҸҜд»ҘеҗҲе№¶гҖӮдҫӢеҰӮпјҢдёҖдёӘдәәжңүеҹәжң¬дҝЎжҒҜе’ҢиҒ”зі»дҝЎжҒҜпјҢдё”з ҒйғҪжҳҜиә«д»ҪиҜҒеҸ·гҖӮ

иҪ¬жҚўеҗҺзҡ„е…ізі»жЁЎејҸеҸҜд»ҘеҗҲе№¶дёәдёҖдёӘпјҡ

  • дәә(иә«д»ҪиҜҒеҸ·, е§“еҗҚ, е№ҙйҫ„, йӮ®з®ұ, з”өиҜқ)

з®Җиҝ°ж•°жҚ®еә“и®ҫи®Ўзҡ„еҹәжң¬жӯҘйӘӨгҖӮ

дё»иҰҒжӯҘйӘӨпјҡв‘ йңҖжұӮеҲҶжһҗв‘ЎжҰӮеҝөжЁЎеһӢи®ҫи®Ўв‘ўйҖ»иҫ‘жЁЎеһӢи®ҫи®Ўв‘Јзү©зҗҶжЁЎеһӢи®ҫи®Ўв‘Өж•°жҚ®еә“зҡ„е®һж–Ҫв‘Ҙж•°жҚ®еә“зҡ„з»ҙжҠӨ
