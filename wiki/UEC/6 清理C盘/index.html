6.清理C盘

我们使用电脑时，C盘总是不知不觉就“变红”。明明下载的东西都改到了其他盘符，但C盘空间还是莫名其妙变少了。

这通常是因为：

  1. 系统备份：频繁的备份文件占用了空间。
  2. 临时文件：软件运行产生的缓存和临时数据。
  3. 默认路径：很多软件即使安装在D盘，其配置文件和缓存数据依然默认保存在C盘（如微信、WPS等）。
  4. 系统垃圾：长期使用堆积的冗余文件。

下面是几种行之有效的清理方法，建议按顺序操作。

一、利用Windows自带的磁盘清理工具

这是最安全、最基础的清理方式，可以删除系统升级留下的旧版本文件。

1、点击C盘右键选择【属性】

2、点击【磁盘清理】

3、关键步骤：点击【清理系统文件】
勾选需要清理的文件（通常“Windows更新清理”会占用很大空间），点击确定即可。

二、存储设置与自动清理

Windows 10/11 自带了非常智能的存储管理功能。

1、在设置中打开系统中的【存储】
按 Win + i 打开设置，进入“系统” -> “存储”。

2、清理临时文件
在存储中找到“临时文件”点进去，勾选并删除不需要的内容。

3、开启自动清理（存储感知）
将【存储感知】设置打开，这样电脑就会在空间不足时自动清理临时文件，省去手动维护的麻烦。

三、更改新内容的保存位置（选做）

为了防止C盘未来继续膨胀，我们可以修改系统默认的下载和安装路径。

操作步骤：
在设置中找到【系统】->【存储】->【更多存储设置】，点击【更改新内容保存位置】。将“新的应用”、“新的文档”等选项修改为其他盘符（如 D 盘），点击【应用】生效。

🛑 避坑指南（重要）：

这个方法仅建议“双硬盘”用户使用（例如你额外加装了一块机械硬盘作为 D 盘）。

现在的笔记本电脑大多只有一块硬盘（只是逻辑上分成了 C 盘和 D 盘）。在这种情况下，把 C 盘的压力转给 D 盘属于“左手倒右手”，实际意义不大，反而可能导致文件目录混乱。

如果你的 C 盘空间不足，而 D 盘还有大量空闲，更推荐直接把 D 盘的空间“划拨”给 C 盘，这才是治本的方法。

👉 具体操作请参考下一章：7. 磁盘分区与磁盘扩容

四、手动清理系统缓存 (%temp%)

Windows 系统中有许多缓存文件，包括浏览器缓存、日志文件等。

可以通过 【Win+R】 打开运行窗口，输入 %temp% 并回车。

弹出的文件夹中全是临时垃圾，按 Ctrl + A 全选，直接删除即可。（如果有文件提示正在使用，跳过即可）。

五、终极清理：WizTree + AI 智能判断

如果做完以上步骤，C盘空间依然紧张，说明有“大文件”或者很多程序的不规范操作导致把垃圾藏在你看不到的深层目录里。

这时我们需要使用 WizTree 进行可视化分析，并配合 AI 来判断能否删除。

1. 为什么推荐 WizTree？

相比于其他的分析软件，WizTree 扫描速度极快（官方宣称世界上最快），因为它直接读取硬盘的主文件表（MFT）。它能通过色块图直观地告诉你，到底是什么文件占用了你的 C 盘。

  • 官网下载：https://diskanalyzer.com/download
  • 费用：个人使用免费

2. 使用方法

第一步：扫描
下拉选择 [C:] 盘，点击扫描。

第二步：查看可视化图表
查看下方的“树状图”（矩形色块）。色块越大，代表文件占用空间越大。

在树状图中点击某个大色块，上方列表会自动定位到该文件所在的文件夹。

3. 核心技巧：配合 AI 清理“未知的大文件”

很多时候我们不敢删 C 盘文件，是因为不知道那个文件夹是干嘛的（比如 AppData 里的各种乱码文件夹）。现在你可以利用 AI 来辅助判断。

操作流程：

  1. 在 WizTree 中发现一个占用巨大的陌生文件夹或文件

  2. 右键点击该文件/文件夹，选择 【复制路径】

  3. 打开你的 AI 助手（如 ChatGPT、Kimi、DeepSeek 等），发送如下指令：

“我的C盘空间不足，发现这个路径下的文件占用了很大空间：[粘贴你刚才复制的路径]。请问这是什么软件的文件？我可以删除它吗？删除了会有什么后果？”

通过这个方法，你通常能揪出以下“隐形杀手”：

  • 游戏录像：NVIDIA 或 OBS 在后台自动录制的高清视频。
  • WPS/Office 备份：囤积了大量历史版本的文档备份。
  • 微信/QQ 缓存：FileStorage 文件夹里存了好几年的图片和视频垃圾。
  • 设计软件缓存：Adobe PR/AE 等软件留下的巨大媒体缓存。

警告：如果您不确定相关文件是否可以删除，且 AI 提示该文件为系统关键文件（如 WinSxS 或 Pagefile.sys），请不要删除。
