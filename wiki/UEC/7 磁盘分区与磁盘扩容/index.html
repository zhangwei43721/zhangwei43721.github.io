7.磁盘分区与磁盘扩容

一、为什么要分区

1. 数据分类管理

不同类型的数据可以存储在不同的分区中，例如：

  • 系统分区：⽤于安装操作系统和相关程序（通常是 C 盘）。
  • 数据分区：⽤于存储⽂档、照⽚、视频等个⼈⽂件。
  • 备份分区：⽤于存储备份⽂件。

这样可以保持数据的整洁，有助于快速查找和管理⽂件。

2. 提⾼数据安全性

  • 如果操作系统所在的分区（如 C 盘）出现问题（如系统崩溃），其他分区中的数据（如 D 盘、E 盘）通常不会受到影响。
  • 系统分区与数据分开可以减少数据丢失的⻛险。

3. 提⾼系统性能

  • 操作系统和常⽤软件放在⼀个分区，减少磁盘碎⽚⽣成的概率，提⾼读写速度。
  • 数据分区的频繁操作不会影响系统性能。

4. 便于重装系统

  • 系统分区和数据分开后，重装系统时只需格式化系统分区，其他分区的数据不会丢失。
  • 这样可以节省备份数据的时间。

5. 多系统需求

  • 分区可以安装多个操作系统（如 Windows 和 Linux），每个系统可以有独⽴的分区，互不⼲扰。

二、如何进⾏分区与扩容

1. 前置操作（关闭 BitLocker）

注意：部分电脑出⼚会⾃动开启 BitLocker，分区会被锁定，导致无法操作。

  1. 按下 Win 键，输入“控制面板”，打开控制面板。

  2. 调整查看方式为“类别”，选择系统和安全。

  3. 选择 BitLocker 加密。

  4. 如果已经启用，那么关闭即可。

2. 开始操作

打开我们的 DiskGenius 软件（我拿到的是一个绿色的压缩包软件，为了方便用，我把他打包为了单文件，所以没有 logo）。

在这个界面我们可以看到我们的相关配置。

3. 分区

  • 如果有空闲的（灰色区域），就直接右键就可以建立新分区。
  • 如果是将磁盘进行拆分分区，那么直接在已经有的磁盘上右键也可以。
  • 新建好后记得点一下左上角保存更改。

4. 扩容（重点：C 盘扩容）

如果是 C 盘快爆炸了，现在最快的急救方法是“拆东墙补西墙”，也就是将其他分区的空间调度过来。

注意：这里说的其他分区要和你的扩容的分区在同一根磁盘上（物理硬件），不然没法将内存移动过去。

我们这里演示扩容一下 F 盘（原理同 C 盘扩容）。

  1. 右键选择扩容分区。

  2. 选中需要移动空间过去的分区（即从哪个盘“割肉”）。

  3. 调整容量：

    • “调整后容量”是指移动空间后，原本的分区的最终容量。
    • “分区后部”（有时是前部，这个取决于相对位置）是指给我们需要扩容的分区多少内存。
磁盘 空间
C（原本） 149.2G
F（原本） 50G
C（扩容后） 149.2 - 67.37 = 81.83G
F（扩容后） 50 + 67.37 = 117.37G

比如我原本 C 是 149.2G，F 是 50G。这里分区完成后 C 就是 81.83G，F 就是 50 + 67.37 = 117.37G。

  1. 然后点击开始就行。

    • 特别注意：如果对 C 盘（系统盘）进行操作，软件会提示你需要重启到 PE 环境（后面都是自动的，不需要管了，等跑完就行了）。
