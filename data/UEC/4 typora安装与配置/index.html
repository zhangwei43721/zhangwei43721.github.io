4.1.Typora安装与配置

一、安装和激活

我们直接将其解压后一路点击 ok 就行了。

关键步骤：将文件夹中的 winmm.dll 移动到 Typora 安装路径即可。

注意：这里大概率会报毒，所以请关闭杀毒软件。

二、插件使⽤

有一个很好用的插件可以优化 Typora 的使用，直接按照官方给出的文档进行安装就行了，我这里也做一下示范。

  • GitHub 项目地址：
    obgnail/typora_plugin: Typora 插件，功能增强工具

1. 基础界面配置

3. 编辑器配置

将成对的符号勾选上，这样我们打一个符号就不用再打第二次，直接进行书写了。换行符因人而异。

4. 图像配置（重要）

图像配置是最重要的（至少对我来说）。

推荐设置：
我们在这里选择插入图片的时候，将其复制到 ./${filename}.assets 文件夹。

  • 这样会在我们 md 文档的同级目录新建一个文件夹（如果没有的话），然后将图片全部备份放一个进去。
  • 这样就有利于我们的文件管理，避免图片找不到。
  • 当然这里我们还可以用图床的方式，详情可以看我另一个视频：【保姆】手把手搭建 hexo 博客与自动部署和图床说明。

注意事项：
特别建议把前面两个给勾选了（对本地位置使用相对路径）。

  • 这样的话，你将 md 和图片压缩包分享给别人的时候，别人可以直接打开 md 进行编辑（因为用的是相对路径，只要 md 文件和 .assets 文件夹在同一个目录下就可以引用）。
  • 避免别人打开的时候还要去改一下图片位置，非常麻烦。
