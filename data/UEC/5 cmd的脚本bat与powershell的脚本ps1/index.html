5.2.CMD脚本(BAT)与PowerShell脚本(PS1)

⼀、BAT ⽂件详解

1. 什么是 BAT ⽂件？

  • 定义：BAT ⽂件（批处理⽂件）是⼀种以 .bat 或 .cmd 作为扩展名的脚本⽂件，基于 Windows 命令提⽰符（CMD）运⾏。
  • 作⽤：
    • 批量执⾏多条 CMD 命令。
    • 实现简单的任务⾃动化。
    • 通常⽤于简单的⽂件操作、系统管理和任务调度。

2. BAT ⽂件的特点

  • 基于纯⽂本命令。
  • 简单易⽤，适合快速搭建脚本。
  • 不⽀持复杂数据处理和对象操作。
  • 仅⽀持 Windows，⽆跨平台能⼒。

3. BAT ⽂件常⽤语法和命令

基本语法

1. 注释：以 :: 或 REM 开头。

2. 变量：

  • 声明：SET 变量名=值
  • 调⽤：%变量名%

3. 条件语句：IF

4. 循环语句：FOR

常⽤命令

命令作用示例
echo输出文本到控制台echo Hello World!
pause暂停，等待用户按键pause
cls清屏cls
dir列出当前目录内容dir
cd切换目录cd C:\Users
del删除文件del file.txt
copy复制文件copy source.txt dest.txt
move移动文件move file.txt folder\
ping测试网络连接ping www.google.com

1. 创建⽂件并写⼊内容

作用：创建 file.txt 并写入一行文本。

2. ⽂件检查

3. 批量重命名⽂件

作用：将当前目录下的 .txt 文件重命名为 renamed_文件名 。

4. ⾃动备份

作用：将 C:\source 文件夹备份到 C:\backup 。

5. 简单菜单

作用：提供一个简单的交互菜单，用户可以选择操作。

⼆、PS1 ⽂件详解

1. 什么是 PS1 ⽂件？

  • 定义：PS1 ⽂件是基于 PowerShell 脚本语⾔的脚本⽂件，以 .ps1 为扩展名。
  • 作⽤：
    • 实现复杂的任务⾃动化。
    • ⽀持跨平台操作（Windows、Linux 和 macOS）。
    • 适⽤于系统管理、⽹络任务、数据处理等⾼级操作。

2. PS1 ⽂件的特点

  • ⾯向对象：输出是 .NET 对象，⽀持操作属性和⽅法。
  • 强⼤的逻辑和模块⽀持：⽀持函数、模块、⾃定义命令等。
  • 跨平台：PowerShell Core（7.x）⽀持多操作系统。
  • 安全性：默认情况下，脚本运⾏受执⾏策略限制（需要调整 ExecutionPolicy）。

注意：因为 PowerShell 默认是没有开启脚本权限的，所以需要先执⾏⼀下 Get-ExecutionPolicy 来看⼀眼权限，然后使⽤ Set-ExecutionPolicy 这个命令来临时给⼀下权限（只在当前窗⼝有效，关闭后需要重新给权限，这样会⽐较安全）。

  • Restricted：默认的设置，不允许任何 script 运⾏。
  • AllSigned：只能运⾏经过数字证书签名的 script。
  • RemoteSigned：运⾏本地的 script 不需要数字签名，但是运⾏从⽹络上下载的 script 就必须要有数字签名。
  • Unrestricted：允许所有的 script 运⾏。

3. PS1 ⽂件常⽤语法和命令

基本语法

1. 注释：

  • 单⾏注释：#
  • 多⾏注释：<# ... #>

2. 变量：

  • 声明：$变量名 = 值
  • 调⽤：直接使⽤ $变量名

3. 条件语句：if-else

4. 循环语句：foreach

常⽤命令

命令作用示例
Write-Output输出文本到控制台Write-Output "Hello World!"
Set-Content创建文件并写入内容Set-Content -Path file.txt -Value "Hello"
Add-Content追加内容到文件Add-Content -Path file.txt -Value "World"
Remove-Item删除文件Remove-Item -Path file.txt
Copy-Item复制文件Copy-Item -Path source.txt -Destination dest.txt
Move-Item移动文件Move-Item -Path file.txt -Destination folder
Test-Path检查文件或路径是否存在Test-Path -Path file.txt
Get-Content显示文件内容Get-Content -Path file.txt

1. 创建⽂件并写⼊内容

作用：创建文件并写入多行内容。

2. 检查⽂件是否存在

作用：检查文件是否存在并输出结果。

3. 批量重命名⽂件

作用：将当前目录下所有 .txt 文件重命名为 new_文件名 。

4. ⾃动备份

作用：将 C:\source 下的所有文件和文件夹递归复制到 C:\backup 。

5. 下载⽂件

作用：从指定 URL 下载文件。

6. 调⽤ REST API

作用：调用 REST API 并处理返回的数据。
