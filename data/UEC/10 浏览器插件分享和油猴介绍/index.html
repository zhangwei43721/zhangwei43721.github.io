10.浏览器插件分享和油猴介绍

一、EDGE 插件 🌐

1. 为什么要使⽤插件？ 🤔

浏览器插件是一种小型软件程序，能为浏览器扩展各种实用功能。合适的插件可以帮你：

  • ⼤幅提升⼯作效率，⾃动化⽇常操作
  • 扩展浏览器原⽣功能
  • 打造个性化的浏览体验
  • 优化⽹⻚浏览体验（如去⼴告、护眼等）

2. 我的常⽤插件推荐 ⭐

  1. AdGuard ⼴告拦截器 - 智能拦截各类⼴告和追踪器，让⽹⻚更加清爽
  2. BewlyBewly - 哔哩哔哩视频增强⼯具，⽀持下载、预览、画质提升等功能
  3. Global Speed - 全⽹视频倍速播放，⽀持快捷键，让学习和观影更⾼效
  4. Tai Sentry - 检测时间
  5. UITab - 个性化新标签⻚，集成实⽤⼯具和⾃定义设置
  6. 猫抓 - 强⼤的媒体嗅探⼯具，轻松下载⽹⻚⾳视频资源
  7. 脚本猫 - 脚本管理⼯具
  8. 超级复制 - ⼀键解除⽹站复制限制，提升⼯作效率

二、精选推荐插件 🔥

1. 翻译⼯具 🌍

这类插件可以帮助你快速理解外语内容，提升阅读效率：

  • DeepL翻译 - 基于 AI 的⾼精度翻译，专业术语翻译更准确
  • 沉浸式翻译 - ⽀持双语对照显⽰，PDF ⽂档翻译，让阅读更流畅
  • 彩云⼩译 - 实时⽹⻚翻译，⽀持划词和全⽂翻译

2. ⽣产⼒⼯具 ⚡

这些工具能帮你更好地管理信息，提升工作效率：

  • Notion Web Clipper - ⼀键保存⽹⻚内容到 Notion，⽀持智能分类
  • OneNote Web Clipper - 将⽹⻚内容剪藏到 OneNote，便于后续整理
  • Grammarly - AI 驱动的英⽂写作助⼿，提供语法检查和写作建议

3. 开发⼯具 💻

面向程序员的效率工具：

  • React Developer Tools - React 应⽤调试利器，实时查看组件结构和状态
  • Vue.js devtools - Vue 开发必备，监控数据流向和组件变化
  • JSON Viewer - 智能格式化 JSON 数据，提升代码可读性
  • WhatFont - ⼀键识别⽹⻚字体，设计参考必备

4. 购物助⼿ 🛒

帮你买到最划算的商品：

  • 历史价格查询 - 显⽰商品历史价格⾛势，避免虚假促销
  • 京价保 - ⾃动监控京东商品价格，降价⾃动提醒
  • ⽐价助⼿ - ⾃动对⽐多平台价格，找到最优惠的选择

5. 标签⻚管理 📑

帮助你更好地组织浏览器标签：

  • OneTab - ⼀键收集所有标签⻚，节省内存占⽤
  • Tab Groups - 按主题分组管理标签，⼯作更有条理
  • Session Buddy - 保存浏览会话，随时恢复⼯作状态

6. 实⽤⼯具 🛠

提升日常使用体验：

  • Dark Reader - 智能⽣成⽹站深⾊模式，保护眼睛
  • 简悦 - 提供沉浸式阅读体验，去除⼲扰元素
  • FireShot - ⽹⻚截图⼯具，⽀持完整⻚⾯截图

三、油猴 & 脚本猫 🔧

1. 什么是油猴和脚本猫？

油猴 (Tampermonkey) 和 脚本猫 (ScriptCat) 是强大的用户脚本管理器，让你能安装各类增强脚本来改善网站体验。

主要功能：

  • 智能去除⼴告和弹窗⼲扰
  • 解锁⽹站⾼级功能
  • 优化⽹⻚显⽰效果
  • 突破各类⽹站限制
  • ⾃动执⾏重复操作

注：脚本猫作为国产脚本管理器，针对中文用户优化，预装常用脚本，性能更优。

2. 我的常⽤脚本

1. AC-baidu 重定向优化

  • 地址： https://greasyfork.org/zh-CN/scripts/14178
  • 功能：优化百度、搜狗、⾕歌搜索结果，去除重定向，添加⽹站图标，⽀持双列展⽰，去除⼴告和商业推⼴。

2. CSDN 优化与美化

  • 地址： https://greasyfork.org/zh-CN/scripts/471071
  • 功能：去除 CSDN ⼴告，优化⻚⾯布局，⾃动展开⽂章，解除复制限制，提供更好的阅读体验。
