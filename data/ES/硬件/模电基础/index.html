模拟电路基础

半导体

半导体指常温下导电性能介于导体与绝缘体之间的材料。常见的半导体材料有硅、锗、砷化镓等，硅是各种半导体材料应用中最具有影响力的一种

导体：铁、铝、铜等金属元素等低价元素，其最外层电子在外电场作用下很容易产生定向移动，形成电流。

绝缘体：惰性气体、橡胶等，其原子的最外层电子受原子核的束缚力很强，只有在外电场强到一定程度时才可能导电

为什么采用半导体材料制作电子元件？

物质的导电性取决于原子结构，而半导体材料的最外层电子既不像导体那样容易摆脱原子核的束缚，也不像绝缘体那样被原子核束缚得那么紧，因而其导电性介于二者之间。在形成晶体结构的半导体中，人为地掺入特定的杂质元素时，导电性能可以有几百万倍的增长，因此导电性能具有可控性；并且在光照和热辐射条件下，其导电性能还有明显的变化；这些特殊性质决定了半导体可以制成各种电子器件

本征半导体

指纯净的具有晶体结构的半导体。晶体中的原子在空间中形成排列整齐的点阵，称之为晶格

共价键：由于相邻原子间的距离很小，因此，相邻的两个原子的一对最外层电子(即价电子)不但各自围绕自身所属的原子核运动，而且出现在相邻原子所属的轨道上，成为共用电子，这样的组合称为共价键结构

自由电子：最外层电子（价电子）由于热运动（热激发）获得足够能量，挣脱共价键的束缚而形成。 此时原子核因失去一个电子而带正电

空穴：自由电子形成后，共价键中留有的一个空余位置。因此，在本征半导体中，自由电子和空穴是成对存在的。若在本征半导体两端外加一电场，一方面自由电子在电场力作用下定向移动，形成电子电流。另一方面，在自由电子移动的过程中，空穴吸引附近的电子来填补，相当于空穴的迁移，而空穴的迁移相当于正电荷的移动，因此可以认为空穴是载流子。（运动是相对的，以自动电子为参照物，空穴向其相反方向移动，可以看成正电荷。）

载流子：指运载电荷的粒子。故在本征半导体中，自由电子和空穴均为载流子

杂质半导体

N型半导体

在纯净的硅晶体中掺入五价元素（如：磷元素），使之代替晶格中硅原子的位置，就形成了N型半导体

由于杂质原子有五个价电子，因此除了形成共价键外，还多出一个电子，此电子不受共价键约束，只要一点能量就能形成自由电子（常温下就能为自由电子）。N型半导体中，自由电子浓度大于空穴浓度，故称自由电子为多数载流子，简称：多子；称空穴为少数载流子，简称：少子；由于杂质原子可以提供电子，故将其称之为施主原子

Negative美/ˈneɡətɪv/：负数的

P型半导体

在纯净的硅晶体中掺入三价元素（如：硼元素），使之代替晶格中硅原子的位置，就形成了P型半导体。

由于杂质原子只有三个价电子，因此在形成共价键是，还多出一个空穴。P型半导体中，空穴浓度大于自由电子浓度，空穴为多子，自由电子为少子；由于杂质原子可以吸收电子，故将其称之为受主原子。

positive美/ˈpɑːzətɪv/正电的

PN结

将P型半导体和N型半导体制作在同一块硅片上，它们的交界处就会形成PN结。

N型半导体是硅原子中掺杂了磷原子，磷原子+5价，自由电子是多子，自由电子带负电，当自由电子离开或者移动时，其所在的原子就变成正离子。

  • 在P区和N区交界处，由于载流子浓度差，N区的自由电子会向P区扩散，P区的空穴会向N区扩散。电子与空穴复合后，N区边界留下了失去电子、无法移动的正离子（施主原子）
  • P区边界留下了得到电子、无法移动的负离子（受主原子）

这些固定的正负离子形成了一个空间电荷区（也叫耗尽层），在此区域内产生了由N区指向P区的“内电场”

如图所示，P区多子是空穴，N区多子是自由电子。

在P区和N区结合时，没有外力作用以及常温条件下，由于分子的扩散运动（物质总是从浓度高的地方向浓度低的地方运动），靠近P区附近的N区自由电子会扩散到P区，那么N区失去电子的那部分区域的原子变成正离子，带正电；而扩散到P区的自由电子会填补空穴，从而让靠近N区的那一部分区域失去了空穴，其原子变成负离子，带负电。

此时，PN交界处就形成了一个内电场。

此时，在 PN结的两端外加电压，就将破坏原来的平衡状态。外加一个电池，且电源正极接N区，电源负极接P区。可以看到，电源正极吸引N区的自由电子，电源负极吸引空穴，从而导致耗尽区扩大，这种现象称之为PN结的反向偏置。

此时将电源的正极接P区，电源的负极接N区，则情况大不一样。电源的负极源源不断地产生自由电子，由于同性相斥，负极产生的自由电子会排挤处于N区的自由电子，迫使N区的自由电子跨过PN结的壁垒，跑到P区，也就是打破内电场，PN结的耗尽区减小，电源的正极吸引电子，从而自由电子就能流动起来，形成电流。这种现象称之为PN结的正向偏置。

二极管

二极管是一种简单的半导体电子元件，具有单向导电性。它由一个PN结组成，即一个P型半导体和一个N型半导体的结合。当电流正向通过时，压降非常小；当电流反相通过时，如果不超过反向电压，几乎是截止状态。

二极管的压降是指当电流通过二极管时，二极管两端的电压降低。这个电压降低的值通常被称为二极管的正向压降或导通压降。二极管的压降是由于电子和空穴在PN结处的重组过程产生的。当电流通过二极管时，电子和空穴在PN结处相遇并复合，释放出能量，这些能量以热能的形式耗散，导致二极管两端的电压降低。硅二极管的正向压降通常在0.6V到0.8V之间。

伏安特性

与PN结一样，二极管具有单向导电性。但是，由于二极管存在半导体体电阻(二极管内部半导体材料本身的电阻)和引线电阻，所以当外加正向电压时，在电流相同的情况下，二极管的端电压大于纯PN结上的压降；或者说，在外加正向电压相同的情况下，二极管的正向电流要小于纯PN结的电流；在大电流情况下，这种影响更为明显。另外，由于二极管表面漏电流(二极管处于反向偏置状态时，仍然有少量电流通过二极管的现象)的存在，使外加反向电压时的反向电流增大。

实测二极管的伏安特性时发现，只有在正向电压足够大时，正向电流才从零随端电压按指数规律增大。使二极管开始导通的临界电压称为开启电压Uon，如图所示。当二极管所加反向电压的数值足够大时，反向电流为Is。反向电压太大将使二极管击穿，不同型号二极管的击穿电压差别很大，从几十伏到几千伏。

发光二极管及其衍生元件

发光二极管（LED）是一种能将电能转化为光能的半导体器件。它具有单向导电性，只有当施加足够大的正向电压和电流时才会发光。发光颜色由其半导体材料决定，常见的有红、绿、黄等。相比普通二极管，LED的开启电压更高，例如红色LED约在1.6V至1.8V之间，绿色LED约为2V。正向电流越大，其发光强度也越强

由于其驱动电压低、功耗小、寿命长、可靠性高等优点，LED及其组合元件（如数码管、点阵屏）被广泛用于各类显示电路中

稳压二极管

稳压二极管是一种硅材料制成的面接触型晶体二极管，简称稳压管。稳压管在反向击穿时，在一定的电流范围内(或者说在一定的功率损耗范围内），端电压几乎不变，表现出稳压特性。因而广泛用于稳压电源与限幅电路之中。

稳压管的伏安特性曲线

稳压器的伏安特性与普通二极管相类似，如图1.2.10(a)所示，正向特性为指数曲线。当稳压管外加反向电压的数值大到一定程度时则击穿，击穿区的曲线很陡，几乎平行于纵轴，表现其具有稳压特性。只要控制反向电流不超过一定值，管子就不会因过热而损坏。稳压管的符号及等效电路如图(b)所示。在等效电路中，二极管D1表示稳压管加正向电压与虽加反向电压但未击穿时的情况，理想二极管、电压源Uz和电阻r的串联支路表示稳压管反向击穿时的等效电路。

稳压管重要参数

1.稳定电压Uz

Uz是在规定电流下稳压管的反向击穿电压。由于半导体器件参数的分散性，同一型号的稳压管的Uz存在一定差别。例如，型号为2CW11的稳压管的稳定电压为3.2~4.5V。但就某一只管子而言，Uz应为确定值。

2.稳定电流Iz

Iz是稳压管工作在稳压状态时的参考电流，电流低于此值时稳压效果变坏，甚至根本不稳压，故也常将Iz记作Izmin。

3.额定功耗Pzm

Pzm等于稳压管的稳定电压Uz与最大稳定电流Izm(或记作Izmax）的乘积。稳压管的功耗超过此值时，会因结温升过高而损坏。对于一只具体的稳压管，可以通过其Pzm的值，求出IzM的值。只要不超过稳压管的额定功率，电流愈大，稳压效果愈好。

4.动态电阻rz

动态电阻rz是稳压管工作在稳压区时，端电压变化量与其电流变化量之比，即rz=△Uz/△Iz。rz愈小，电流变化时Uz的变化愈小，即稳压管的稳压特性愈好。对于不同型号的管子，rz将不同，从几欧到几十欧。对于同一只管子，工作电流愈大，rz愈小。

5.以1N4733A为例

  • 标称稳定电压 (Vz): 5.1V (在49mA电流下测得)
  • 额定工作电流 (Izt): 49mA (这是一个测试电流，也是一个理想的工作点参考)
  • 最小工作电流 (Izk): 1.0mA (低于此电流，稳压效果会急剧变差，此时电阻为550Ω)
  • 动态电阻 (Zzt): 7.0Ω (在49mA电流下的值，是评估其性能的主要参数)
  • 反向漏电流 (Ir): 10μA (在1.0V反向电压下)

由于稳压管的反向电流小于Izmim时不稳压，大于Izmax时会因超过额定功耗而损坏，所以在稳压管电路中必须串联一个电阻来限制电流，从而保证稳压管正常工作，故称这个电阻为限流电阻。只有在R取值合适时，稳压管才能安全地工作在稳压状态。

解答:

稳压二极管应用电路

稳压二极管工作在反向击穿状态，当电路中的输入电压超过5.1V时，比如说15V，此时稳压二极管开始工作，处于导通状态，利用当PN结反向击穿时两端电压保持不变的特性，使其输出电压保持在约5V，保护后续的电路不会因为电压过高而损毁元器件。里面的电阻元器件是起到限流作用，因为稳压二极管的电流是几十毫安，电流过大会损坏稳压二极管。

此时发现，输出电压稳定在5V，但是带负载能力很差，因为没有输出电流。

此时需要对电路进行改进，15V的电压经过配置电阻之后，稳压二极管导通，稳定B点的电压为5.6V，三极管导通，BE之间的压降为0.6V,所以输出电压为5V，三极管还具有放大电流作用，CE之间的电流作为输出电流，此时带负载能力变强。

三极管

晶体管，又名三极管，其实就是连续的两个PN结接在一起，有NPN和PNP两种结构。

我们以NPN型为例，讲解一下三极管导通的原理。

NPN型三极管

我们实际应用中，模拟电路主要是使用三极管的放大原理，数字电路主要是使用三极管的开关原理。

开关特性

我们可以使用一个类比的例子去理解三极管的开关作用。B基极相当于阀门，当B基极的电压为高电平（也就是阀门打开），则C集电极和E发射极导通，也就是C水箱里面的水会流向E出水口。当B基极的电压为低电平（也就是阀门关闭），则C水箱里面的水不能流向E出水口，三极管处于关闭状态。

刚才我们提到，三极管实际上是由两个PN结接在一起的，那么三个区的大概结构图如下，不过实际上每个区的浓度和厚度是不一样的。

从下图可以看到，NPN型三极管的基本构成结构，E发射区的自由电子浓度很高，B基区主要是空穴，但是浓度低，而且很薄，C集电区主要是自由电子，但是浓度低。B区和E区组成一个PN结，B区和C区组成一个PN结，当外界一个电池时，而且电池的正极在E区，电池的负极在C区。电池工作时，BC组成的PN结导通，BE组成的PN不能导通，导致三极管处于截止状态。

此时将电池反接，可以看到，BE组成的PN结导通，BC组成的PN结不导通，导致三极管还是处于截止状态。

因此，为了让三极管导通，我们可以再给BE区接上一个电源，那么电池的负极不断地产生很多电子，移动到E发射区，本身E发射区就有很多电子了，同性排斥会迫使E发射区的电子移动到B基区，打破PN结的势垒。由于B基区很薄，而且空穴浓度低，所以电子会溢出，一部分被正极吸引，跑到电池的正极，从而形成上方的BE回路；另一部分扩散到C区，打破BC的PN结势垒，C集电区的自由电子浓度低，可以承载很多电子，下方的电源正极同时也吸引电子，电子最终流向到下方电源的正极，下方也形成回路，从而整个三极管处于导通状态。

三极管的这种特性，可以作为开关电路的控制端使用

开关特性应用电路

Ø NPN型三极管：如果我们给B极一个5V信号，LED通路就通了，此时晶体管起到了类似继电器的作用。

Ø PNP型三极管：如果我们给B极一个低信号，LED通路就通了。

放大特性

1.截止状态

可以把NPN型三极管想象成下面的水管，B基极就是一个联动阀门。当Vbe的电压小于0.5V时，不能打破PN结的势垒（通常PN结的势垒在0.6~0.7V），所以三极管处于截止状态，不能导通。

2.放大状态

当Vbe的电压大于0.5V时，BE就有电流，阀门被打开一个小口，如果Vbe的电压增加，BE电流变大，那么阀门的开口就越大。由于这是一个联动阀门，所以上方CE之间也被打开，那么CE之间也会有电流，BE之间的电流是小电流，水管直径比较小，联动控制上方的CE之间的电流是大电流，水管直接比较大，这就是利用小电流控制大电流的原理。此时三极管处于放大状态。

3.饱和状态

当Vbe的电压继续增加时，会导致电流增大，阀门持续打开直到完全打开；当Vbe约等于0.7V时，阀门完全打开，无论怎么增加Vbe的电压，都不能使BE电流增加，也就是CE电流也不能继续增加了，此时处于饱和状态。

MOS管

英文全称是Metal Oxide Semiconductor Field Effect Transistor（金属氧化物半导体场效应晶体管）。场效应管也是一种晶体管，它通过电场来控制电流。传统的晶体三极管通过电流来控制电流。场效应管有许多应用，包括放大、开关和电压控制等。场效应管是芯片的底层原理，是芯片中的最小单位。

工作原理

可以看到，MOS管分为三个极，分别是栅极、源极、漏极。当给栅极一个高电平时，MOS管的源极和漏极之间就导通。如图中的橙色实线。

当给栅极一个低电平时，MOS管的源极和漏极之间不导通，MOS管截止。

这是MOS管的内部结构图。

在源极和漏极之间添加一个电源，MOS管是不导通的，因为P区中大部分是空穴，自由电子的浓度很低，无法形成一条让自由电子流动的通路。

此时将栅极添加到电源的正极，这样栅极所连带的金属板就会产生一个电场，P区中离散的自由电子就会在电场的作用下被吸引过来，那么电池的负极源源不断地产生电子，进入到下方的N区，打破PN结的势垒之后，同性相斥，此时电子经过P区，又经过上方的N区，最后电源的正极吸引电子，最终电子流向电源的正极，形成了一条回路。

当将栅极的电压撤去的时候，P区中的自由电子就不会聚集，那么自由电子无法通过P区，MOS管就截止了。

基本放大电路

放大现象存在于各种场合，例如，利用放大镜放大微小物体，这是光学中的放大;利用杠杆原理用小力移动重物，这是力学中的放大;利用变压器将低电压变换为高电压，这是电学中的放大。研究它们的共同点，一是都将原物形状或大小的差异按一定比例放大了，二是放大前后能量守恒，例如，杠杆原理中前后端做功相同，理想变压器的原、副边功率相同等。

利用扩音机放大声音，是电子学中的放大，其原理框图如图所示。

图中+V为供电电源,“丄”为电路的公共端。话筒(传感器)将微弱的声音转换成电信号，经放大电路放大成足够强的电信号后，驱动扬声器(执行机构)，使其发出较原来强得多的声音。电子电路放大的基本特征是功率放大，即负载上总是获得比输入信号大得多的电压或电流，有时兼而有之。能够控制能量的元件称为有源元件，因而在放大电路中必须存在有源元件，如晶体管和场效应管等

**放大的前提是不失真，即只有在不失真的情况下放大才有意义。**晶体管和场效应管是放大电路的核心元件，只有它们工作在合适的区域(晶体管工作在放大区、场效应管工作在恒流区)，才能使输出量与输人量始终保持线性关系即电路才不会产生失真。

由于任何稳态信号都可分解为若干频率正弦信号(谐波)的叠加，所以放大电路常以正弦波作为测试信号。

3.基本共射放大电路

1.电路图

2.电路各元件的介绍

基极电源Vbb:为输入回路提供电源，使晶体管be之间的电压Ube大于发射结的开启电压Uon，并与基极电阻Rb共同决定基极电流Ib。

输入电压ui：正弦波电压，作为输入的信号源，也就是你要放大的信号源。

基极电阻Rb：该电阻的大小控制基极的电流Ib。

晶体管T: 是放大电路的核心元件，将输入信号ui放大成uo。要保证集电结反偏,发射结正偏,使晶体管工作在放大区。

集电极电源Vcc：Vcc应该要足够大，使晶体管的集电结反向偏置，以保证晶体管工作在放大状态，集电极电流Ic = β*Ib。

集电极电阻Rc：将放大电流的变化转换为放大电压的变化，流经集电极电阻上的电流为Ic,所以其电压为IcRc ,从而确定CE之间的电压为 Uce = Vcc-IcRc。

输出电压uo：为放大之后的输出电压。

3.工作原理和波形分析

在放大电路中，当有信号输入时，交流量与直流量共存。当输入电压ui为0时，称放大电路处于静态。将输入信号为零、即直流电源单独作用时晶体管的基极电流Ib、集电极电流Ic、be间电压Ube、管压降Uce称为放大电路的静态工作点Q。常将四个物理量记作Ibq、Icq、Ubeq、Uceq。在近似估算中常认为Ubeq为已知量，对于硅管，取Ubeq为0.6V至0.8V中的某一值，如0.7V。

当有输入电压ui时，基极电流Ib是在原来直流分量Ibq的基础上叠加一个正弦交流ib，因而基极总电流iB = Ibq+ib。根据晶体管基极电流对集电极电流的控制作用（Ic = β*Ib），集电极电流也会在直流分量Icq的基础上产生一个正弦交流电流ic,而且ic = βib ,集电结总电流 ic = Icq + βib。CE之间的交流电压为 uce = Vcc - ic * Rc 。当ic增加时，Rc两端电压变大，则uce减小，所以uce的交流电压跟ic的变化方向正好相反（180度反相）。那么CE之间的总电压 Uce = 直流电压uceq + 交流电压uce 。最终放大电路输出的电压uo 是 Uce中的交流电压uce。
